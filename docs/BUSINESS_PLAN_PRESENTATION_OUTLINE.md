## PRESENTATION DESIGN GUIDELINES

### Visual Style & Branding

**Color Palette:**
- Primary: Forest green (#2C5F2D) - represents trails and nature
- Secondary: Maple red (#D32F2F) - Canadian identity
- Accent: Sky blue (#1976D2) - outdoor recreation
- Neutral: Gray (#546E7A), White (#FFFFFF)

**Typography:**
- Headers: Bold, sans-serif (Montserrat or similar)
- Body: Clean, readable sans-serif (Open Sans or similar)
- Numbers: Monospace for financial data clarity

**Layout Principles:**
- **One idea per slide:** Avoid information overload
- **Visual hierarchy:** Large numbers, bold headers, concise body text
- **White space:** Clean, uncluttered design
- **Consistency:** Use template for all slides

**Image Guidelines:**
- **Hero images:** High-quality trail photos (mountain biking, hiking)
- **Canadian branding:** Subtle 🍁 maple leaf in corner of slides
- **People photos:** Show diverse trail users and volunteers
- **Screenshots:** Use actual TrailLensHQ product screenshots (when available)

### Data Visualization Standards

**Charts & Graphs:**
- **Bar charts:** For comparisons (revenue growth, customer acquisition)
- **Line charts:** For trends over time (ARR growth, cash flow)
- **Pie charts:** For composition (revenue mix, use of funds)
- **Funnels:** For conversion metrics (TAM → SAM → SOM)
- **Gauges:** For single metrics (LTV:CAC ratio, Rule of 40)

**Tables:**
- Limit rows to 5-7 maximum
- Highlight key numbers with color or bold
- Use alternating row colors for readability
- Include units and context

**Icons:**
- Use consistent icon set (Font Awesome or similar)
- Simple, recognizable icons
- Color-coded by category

### Slide Animation & Transitions

**Minimize animations:**
- Simple fade transitions between slides
- No flying text, spinning graphics, or distracting effects
- Focus on content, not animation

**Build reveals:**
- Use bullet point builds for complex slides
- Reveal financial data incrementally to tell a story
- Build comparison tables column by column

### Speaker Notes Strategy

**Each slide should include:**
- Key talking points (3-5 bullet points)
- Statistics and data sources for reference
- Anticipated questions and answers
- Transition language to next slide

**Timing guidance:**
- Title/intro slides: 30-60 seconds
- Problem/solution slides: 2-3 minutes each
- Financial slides: 3-5 minutes (critical for investor understanding)
- Risk/mitigation slides: 2-3 minutes
- Appendix slides: Reference only, 1-2 minutes if addressed
